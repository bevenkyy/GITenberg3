## Usage

```sh
j: scroll down
k: scroll up
h: go to parent dir
l: go to child dir

g: go to top
G: go to bottom

.: toggle hidden files
/: search

down:  scroll down
up:    scroll up
left:  go to parent dir
right: go to child dir

enter: go to child dir
```
